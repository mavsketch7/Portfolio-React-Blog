# Contexto del Proyecto

- Tech Stack: Next.js 14+ (App Router), React, Tailwind CSS, TypeScript.
- Estilo Visual: Minimalista, modo oscuro por defecto, animaciones sutiles (Framer Motion si aplica).
- Estructura: Componentes atómicos en `/components`, páginas en `/app`.
- Regla: Código limpio, tipado estricto en TS y clases de Tailwind combinadas con `cn()` (clsx/tailwind-merge).
